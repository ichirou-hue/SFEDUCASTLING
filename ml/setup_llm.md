# Дообучение LLM на шахматных данных

## Железо: 2x RTX 3090 (48GB VRAM), 700GB RAM
## Что поместится: модели до 13B параметров через QLoRA

## Шаг 1: Выбор базовой модели
- Mistral 7B (рекомендуется — лучшее качество на 7B)
- LLaMA 3 8B
- Qwen 2.5 7B

## Шаг 2: Подготовка датасета
Формат: JSONL, каждая строка:
{
  "instruction": "Оцени шахматную позицию: rnbqkbnr/pppppppp/8/8/4P3/8/PPPP1PPP/RNBQKBNR b KQkq e3 0 1",
  "output": "Белые сыграли 1.e4 — королевский дебют. Пешка контролирует центральные поля d5 и f5. Позиция открытая, стадия — дебют. Рекомендуется ответить 1...e5 (симметричная игра) или 1...c5 (сицилианская защита)."
}

## Шаг 3: Где взять данные
1. Lichess Open Database (миллионы партий с оценками)
   https://database.lichess.org/
2. Stockfish оценки — прогнать позиции через Stockfish, получить eval
3. Ручная разметка — написать 500-1000 примеров анализа позиций
4. Генерация через GPT/Claude — сгенерировать анализы для позиций

## Шаг 4: Обучение через QLoRA
Инструменты: unsloth или axolotl

## Шаг 5: Инференс
Через vLLM или llama.cpp для быстрого ответа
